Install nodemon:

    $ sudo npm i -g nodemon
    
Install dependencies:

    $ npm i
    
Run project:

    $ npm start
    Go http://localhost:3000/
            or
    Make GET,POST requests to the API.

Run tests:

    $ npm test
    
Nodeenv:

    $ nodeenv env
    $ . env/bin/activate
    (env)$ npm test
    (env)$ npm start   

Environment:

    $ node -v
        v.6.9.1
    $ npm -v
        3.10.8
